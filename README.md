## PaperPal

### LangFuse

- Install LangFuse Docker:
  
  ```bash
  $ git clone https://github.com/langfuse/langfuse.git
  $ cd langfuse
  $ docker compose up
  ```

- To run the container:
  
  ```bash
  $ cd langfuse
  $ docker compose start
  ```

- Now, hit `http://localhost:3000` in your browser to access the dashboard of LangFuse.

- On LangFuse dashboard, I created the following:
  
  - Organization: `sanjays_org`
  
  - Project Name: `paper-pal`
  
  - API Key: `<langfuse-paper-pal-api-key>`
